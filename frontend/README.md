# Ram Transport React

This frontend has been converted from Angular to React with Vite.

## Run the React frontend

1. Open a terminal in this folder:
   `C:\Users\HP\OneDrive\Desktop\AnToReact\main\Transport-Frontend-main`
2. Install packages:
   `npm install`
3. Start the React app:
   `npm run dev`
4. Open the URL shown in the terminal, usually:
   `http://localhost:5173`

## Run the backend API

The React app still calls:
`http://localhost:5000/api/v1`

Start the backend from:
`C:\Users\HP\OneDrive\Desktop\AnToReact\main\Transport_Management_System-main`

Run:
`npm install`

Then:
`npm run dev`

If `npm run dev` is not available on your machine, use:
`npm start`

## Build for production

Run:
`npm run build`
